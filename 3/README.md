# Pixel Runner

Pixel Runner — мини-игра в жанре платформер, выполненная на HTML, CSS и JavaScript с использованием HTML5 Canvas API.

## Описание проекта

Игрок управляет персонажем, который должен пройти уровень, собрать все кристаллы и добраться до финиша. На пути расположены платформы и опасные препятствия в виде шипов. Если персонаж касается шипов или падает вниз, игра завершается поражением.

## Функционал

В игре реализованы:

- движение персонажа влево и вправо;
- прыжок с помощью клавиш Space, W или стрелки вверх;
- гравитация и физика падения;
- коллизии с платформами;
- сбор кристаллов;
- система подсчета очков;
- препятствия в виде шипов;
- условие победы;
- условие поражения;
- кнопки управления: Старт, Пауза, Рестарт;
- плавная отрисовка через requestAnimationFrame.

## Управление

- A или стрелка влево — движение влево;
- D или стрелка вправо — движение вправо;
- W, Space или стрелка вверх — прыжок.

## Структура проекта

Проект расположен в папке `3`.

Структура файлов:

- `index.html` — основная HTML-страница игры;
- `README.md` — описание проекта;
- `css/style.css` — стили интерфейса;
- `js/main.js` — главный файл инициализации игры;
- `js/Game.js` — основной класс игры;
- `js/Player.js` — класс персонажа;
- `js/Platform.js` — класс платформ;
- `js/Collectible.js` — класс собираемых предметов;
- `js/Hazard.js` — класс препятствий;
- `js/Collision.js` — модуль обработки коллизий;
- `js/InputHandler.js` — класс обработки пользовательского ввода.

## Реализация

В проекте используется модульная архитектура. Каждый основной объект игры вынесен в отдельный класс.

`Game` — основной класс игры, управляет игровым циклом, состоянием игры, очками и условиями победы или поражения.

`Player` — класс персонажа, отвечает за движение, прыжок, гравитацию и отрисовку игрока.

`Platform` — класс платформ, по которым перемещается персонаж.

`Collectible` — класс собираемых предметов, за которые начисляются очки.

`Hazard` — класс препятствий, при столкновении с которыми игрок проигрывает.

`Collision` — модуль проверки и обработки столкновений.

`InputHandler` — класс обработки пользовательского ввода с клавиатуры.

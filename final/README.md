# P&Co Store

P&Co Store — финальный проект по дисциплине «Веб-дизайн», выполненный по теме «E-commerce с корзиной».

## Описание проекта

Проект представляет собой интернет-магазин одежды P&Co. На сайте реализован каталог товаров, фильтрация по категориям, интерактивная корзина, промокоды, расчет итоговой стоимости и формирование JSON-объекта заказа для передачи на бэкенд.

## Функционал

В проекте реализованы:

- каталог товаров;
- фильтрация товаров по категориям;
- добавление товаров в корзину;
- визуальное изменение кнопки после добавления товара;
- изменение количества товаров в корзине;
- удаление товаров из корзины;
- очистка корзины;
- расчет стоимости товаров;
- расчет скидки;
- расчет доставки;
- применение промокодов;
- сохранение корзины в LocalStorage;
- форма оформления заказа;
- валидация данных покупателя;
- формирование JSON-заказа;
- копирование JSON в буфер обмена.

## Промокоды

В проекте доступны промокоды:

- PANDCO10 — скидка 10%;
- STUDENT15 — скидка 15%;
- FREESHIP — бесплатная доставка.

## Структура проекта

Проект расположен в папке `final`.

- `index.html` — основная страница интернет-магазина;
- `css/style.css` — стили проекта;
- `data/products.js` — массив товаров;
- `js/main.js` — главный файл приложения;
- `js/Cart.js` — логика корзины;
- `js/ProductRenderer.js` — отрисовка карточек товаров;
- `js/PromoService.js` — работа с промокодами;
- `js/Checkout.js` — оформление заказа и формирование JSON;
- `js/StorageService.js` — работа с LocalStorage;
- `assets/images/` — изображения товаров и hero-блока.

## Использованные технологии
